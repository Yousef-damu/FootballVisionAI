## 今日の目標
- YOLOを導入する
- 静止画から選手を検出する
- AIによる物体検出の仕組みを理解する

## 実施内容
- Ultralytics YOLOをインストール
- YOLOv8nモデルをダウンロード
- 静止画を入力として物体検出を実行
- サッカー画像から選手とボールの検出に成功
- YOLOの推論結果を画像として表示

## 発生した問題

### 1. YOLOモデルのダウンロード警告
**エラー**
```
WARNING Download failure, retrying...
```

**原因**
SSL証明書の確認時に一時的な警告が発生した。

**解決**
自動で再試行され、正常にYOLOモデルがダウンロードされた。

**学んだこと**
警告が表示されても、内容を確認してから判断することが重要である。

### 2. Image Read Error
**エラー**
```
WARNING Image Read Error
IndexError: list index out of range
```

**原因**
OpenCVが画像ファイルを正しく読み込めなかった。

**解決**
画像をPNG形式で保存し直したところ、正常に読み込めた。

**学んだこと**
AIのエラーではなく、入力データに原因がある場合も多い。まず入力データを確認することが重要である。

### 3. YOLOが不要な物体も検出した
**状況**
選手やボール以外の物体も検出された。

**原因**
YOLOはCOCOデータセットで学習しており、人物・椅子・スポーツ用品など80種類の物体を検出するため。

**今後の対応**
研究では選手とボールのみを抽出する処理を追加する。

## 今日学んだこと
- YOLOは画像内の物体を検出するAIモデルである。
- YOLOは画像を理解しているのではなく、物体の位置・種類・信頼度を推定している。
- 出力される情報は「クラス」「信頼度」「Bounding Box」である。
- AIの性能を評価する前に、入力データが正しいか確認することが重要である。
