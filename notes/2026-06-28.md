## 発生した問題

### 1. destroyALLWindowsエラー
**エラー**
```
AttributeError: module 'cv2' has no attribute 'destroyALLWindows'
```

**原因**
関数名を `destroyALLWindows()` と書いていた。

**解決**
`destroyAllWindows()` に修正した。

**学んだこと**
Pythonでは大文字・小文字が区別されるため、関数名は正確に書く必要がある。

### 2. 動画がすぐ終了した
**原因**
動画を最後まで正常に読み込んで終了していた。

**解決**
while文を一旦外し、最初のフレームだけ表示して動作確認を行った。

**学んだこと**
動画はFrame（画像）の集合であり、Computer VisionではFrame単位で処理を行う。

### 3. pythonコマンドが見つからない
**エラー**
```
python: command not found
```

**原因**
Mac環境では `python3` を使用する設定になっていた。

**解決**
仮想環境内のPythonを使用するようにした。

**学んだこと**
実行するPythonがどこにあるかを確認することが重要である。
